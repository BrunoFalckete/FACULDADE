Callbacks

Callbacks são funções que são passadas como argumentos para outras funções e executadas posteriormente. É uma técnica fundamental em JavaScript para lidar com operações assíncronas e eventos.

Aplicações:

- gateways de pagamento
- aplicações mobile

Vantagens:

- Controle de fluxo assíncrono
- Flexibilidade

Desvantagens:

- Callback Hell
- Bloqueio

Em resumo, a programação com callbacks é uma abordagem valiosa para lidar com operações assíncronas, permitindo a execução de código não bloqueante. No entanto, é crucial adotar práticas que evitem o callback hell e compreender os desafios associados à sua utilização para garantir um código mais legível, manutenível e eficiente.

Exemplo de Código

            
                // Aguarda o carregamento completo do DOM antes de executar o código
                document.addEventListener("DOMContentLoaded", () => {
                    // Obtém referências aos elementos do DOM
                    const button = document.getElementById("myButton");
                    const output = document.getElementById("output");
                    const inputValue = document.getElementById("inputValue");

                    // Adiciona um ouvinte de eventos para o clique no botão
                    button.addEventListener("click", () => {
                        // Obtém o valor inserido pelo usuário
                        const userValue = inputValue.value;
                        
                        // Chama a função assíncrona passando o valor e uma função de callback
                        realizarOperacaoAssincrona(userValue, (resultado) => {
                            // Atualiza o conteúdo do elemento de saída com o resultado
                            output.textContent = `Resultado da operação: ${resultado}`;
                        });
                    });

                    // Função que simula uma operação assíncrona
                    function realizarOperacaoAssincrona(valor, callback) {
                        // Simula uma operação assíncrona (por exemplo, uma requisição AJAX)
                        setTimeout(() => {
                            // Gera um resultado aleatório com base em uma condição
                            const resultado =
                                Math.random() > 0.5 ? `Sucesso com ${valor}` : `Erro com ${valor}`;
                            // Chama a função de callback passando o resultado
                            callback(resultado);
                        }, 2000);
                    }
                });
